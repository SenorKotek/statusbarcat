# statusbar_cat

`statusbar_cat` — маленькая macOS status bar утилита с детализированным ASCII-котом.

## Что умеет
- сидит в status bar и анимируется в реальном времени;
- следит за курсором (взгляд влево/вправо/по центру);
- периодически моргает;
- виляет хвостом;
- периодически высовывает язык;
- при вводе с клавиатуры моргает и последовательно топает передними лапками.

## Локальный запуск (macOS 13+)
```bash
swift run ayabar
```

## Сборка `.app`
```bash
./scripts/package_app.sh
open build/statusbar_cat.app
```
